Concept 21 — Notion-Style Sidebar

Clean, minimal tree navigation with hover-reveal controls

Inspired by Notion's navigation paradigm. Features a collapsible tree structure with smooth expand/collapse animations, breadcrumb trail, and hover-reveal action buttons on each item.

  • Resizable sidebar with drag handle (grab the right edge)
  • Collapsible tree sections with toggle arrows
  • Colored icon squares for visual categorization
  • Hover-reveal action buttons and item counts
  • Breadcrumb trail showing current location
  • Search bar with keyboard shortcut hint
  • User profile with settings in sidebar footer
  • Smooth dark/light theme transition
  • Items indent slightly on hover
